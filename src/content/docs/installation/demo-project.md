# Quick Setup & Demo Project

Just want to get started quickly? Try out a demo project from the Endless Foundation's curriculum!

1. Download Moddable Platformer or Threadbare from the [latest release](https://github.com/inkandswitch/backstitch-godot-plugin/releases/latest)
2. Unzip the file
3. Double-click the launch script for your platform:
   - Windows: `backstitch-launcher.windows.exe`
   - Linux: `backstitch-launcher.linux`
   - macOS: `backstitch-launcher.macos`

<div class="warning">

### Note

Moddable Platformer and Threadbare both use Backstitch's [alpha sync server](../server/alpha-server.md). Please do not rely on this server to store your data, as we might need to reset it at any time – make sure you always have a local copy of your project. Additionally, anyone that knows your Project ID could access your project, so please do not store any sensitive or proprietary data.

</div>
