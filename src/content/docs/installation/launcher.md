# Using Backstitch Launcher

The easiest way to install Backstitch into an existing project is to use the [Backstitch Launcher](https://github.com/inkandswitch/backstitch-launcher). This launcher...

- Automatically downloads the correct version of Godot and Backstitch
- Launches your project with Godot

## Installing Backstitch Launcher

1. Download the [latest release of backstitch-launcher](https://github.com/inkandswitch/backstitch-launcher/releases/latest).
2. Unzip the folder, and place its contents in the **root directory of your Godot project** (next your `project.godot` file).
3. If you use Git, merge the `.gitignore.template` with your own `.gitignore`.
4. Run `backstitch-launcher` for your platform by double-clicking the file:
   - Windows: `backstitch-launcher.windows.exe`
   - Linux: `backstitch-launcher.linux`
   - macOS: `backstitch-launcher.macos`
