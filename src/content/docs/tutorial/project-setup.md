# Project & Server Setup

When you launch a project with Backstitch, a new sidebar appears alongside the inspector. The Backstitch Sidebar is where you'll interact with all of the plugin's functionality.

![The Backstitch sidebar](sidebar.png)

Before continuing, you'll need to pick a server. You can either use our [Alpha Test Server](../../server/alpha-server.md) or [host your own server](../../server/host.md). Alternatively, you may simply proceed offline with no server selected, but you won't get any live syncing features with collaborators.

![Selecting a server](server-selection.png)

Once you have selected your server, you need to create or join a Backstitch Project. To create a project, simply click the "create a new project" button.

!["Create a new project" button](create-project.png)

When you create a project, your entire Godot project will be uploaded into Backstitch, and synced with the server you selected. It might take some time, so be patient!

Alternatively, if you have a Project ID that someone has shared with you from the same server, you can [load their project instead](../sharing.md).
