# Alpha Test Server

Backstitch provides an Alpha Test Server at `tcp://alpha.backstitch.dev:8085`. Feel free to use it for testing, demonstration, and classroom purposes, but please do **not** store sensitive or proprietary data on the server. There is currently no authentication, so if someone finds or guesses your Project ID, they can access all your data!

We also reserve the right to shut down the alpha server at any time, or wipe _all_ of your data. So, while using it, make sure to always keep both local and remote backups with other software (like Git).
