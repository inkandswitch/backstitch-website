# Troubleshooting

Backstitch is still experimental, and errors can happen. Please [join our Discord server](https://discord.gg/SkW9vem5Ez) for support! We'd be happy to help.

As long as you're regularly saving your work, data loss is unlikely. All of your changes are still being saved to your computer's local filesystem.

If syncing is not working, you can try falling back to other methods of syncing data, like using a Git repository.

If Godot crashes or seems unresponsive, try restarting the editor.

## Limitations

Backstitch is still in early alpha, and only supports projects up to several gigabytes (depending on your system's memory). We have plans to support larger projects, but for now, Backstich is best for smaller games, game jams, and classrooms!
