# Troubleshooting

Backstitch is still experimental, and errors can happen.

As long as you're regularly saving your work, data loss is unlikely. All of your changes are still being saved to your computer's local filesystem.

If syncing is not working, you can try falling back to other methods of syncing data, like using a Git repository.

If Godot crashes or seems unresponsive, try restarting the editor.

## Limitations

Backstitch is currently limited in the project sizes it can support. It is suitable for projects up to several gigabytes in size, depending on your system configuration. However, since all files will be stored in your system's RAM, large projects will degrade system performance with Backstitch.

We have long-term plans to reduce Backstitch's memory usage and improve its performance!
