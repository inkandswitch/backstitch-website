# Servidor de pruebas alfa

Backstitch ofrece un servidor de pruebas alfa en `tcp://alpha.backstitch.dev:8085`. Puedes usarlo libremente para pruebas, demostraciones y fines docentes, pero por favor **no** almacenes datos sensibles ni propietarios en él. Actualmente no hay autenticación, así que si alguien encuentra o adivina el ID de tu proyecto, ¡podrá acceder a todos tus datos!

También nos reservamos el derecho de apagar el servidor alfa en cualquier momento, o de borrar _todos_ tus datos. Por eso, mientras lo uses, asegúrate de mantener siempre copias de seguridad locales y remotas con otro software (como Git).
