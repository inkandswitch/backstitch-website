# Hospedar un servidor

Como alternativa más estable y segura, puedes hospedar un servidor de Backstitch en tu red local o en internet abierto. Encontrarás las instrucciones en el repositorio [Backstitch Sync Server](https://github.com/inkandswitch/backstitch-sync-server).

<div class="warning">

### Nota

El servidor de sincronización de Backstitch no proporciona autenticación por ahora. Esto significa que cualquiera puede acceder a tus datos si adivina el ID del proyecto. Si expones este servidor a internet abierto, es **muy recomendable** usar un túnel VPN con la autenticación que prefieras u otro esquema de autenticación.

</div>
