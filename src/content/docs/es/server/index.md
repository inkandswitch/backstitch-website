# Servidor de sincronización

Para colaborar con otras personas, necesitarás unirte a un servidor de sincronización o configurar uno.

- Si lo deseas, puedes utilizar nuestro [servidor de pruebas alfa](./alpha-server.md).
- Para mayor seguridad, puedes [hospedar tu propio servidor](./host.md) si tienes algo de soltura con la línea de comandos.
