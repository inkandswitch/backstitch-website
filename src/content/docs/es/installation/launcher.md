# Usar el Backstitch Launcher

La forma más sencilla de instalar Backstitch en un proyecto existente es usar el [Backstitch Launcher](https://github.com/inkandswitch/backstitch-launcher). Este launcher...

- Descarga automáticamente la versión correcta de Godot y de Backstitch
- Inicia tu proyecto con Godot

## Instalar Backstitch Launcher

1. Descarga la [última versión de backstitch-launcher](https://github.com/inkandswitch/backstitch-launcher/releases/latest).
2. Descomprime la carpeta y coloca su contenido en el **directorio raíz de tu proyecto de Godot** (junto a tu archivo `project.godot`).
3. Si usas Git, fusiona la plantilla `.gitignore.template` con tu propio `.gitignore`.
4. Ejecuta `backstitch-launcher` correspondiente a tu plataforma haciendo doble clic en el archivo:
   - Windows: `backstitch-launcher.windows.exe`
   - Linux: `backstitch-launcher.linux`
   - macOS: `backstitch-launcher.macos`
