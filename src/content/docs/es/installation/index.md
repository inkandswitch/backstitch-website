# Configuración e instalación

Backstitch todavía está esperando a que Godot incorpore algunos cambios, así que por ahora dependemos de una **compilación personalizada de Godot 4.7**. En un futuro próximo podremos publicar en la Asset Library, pero de momento, configurar Backstitch requiere un poco de trabajo extra.

- **Inicio rápido**: Si solo quieres descargar un proyecto para ver si te gusta, [prueba Moddable Platformer, Moddable Pong o Threadbare](./demo-project.md) del plan de estudios de la Endless Foundation.
- Para configurar Backstitch en tu propio proyecto, lo más sencillo es [usar el Backstitch Launcher](./launcher.md).
- Si prefieres no usar el launcher, puedes [configurar Backstitch manualmente](./manual-setup.md).
