# Configuración rápida y proyecto de demostración

¿Solo quieres empezar rápido? ¡Prueba un proyecto de demostración del plan de estudios de la Endless Foundation!

1. Descarga un proyecto inicial desde la [última versión](https://github.com/inkandswitch/backstitch/releases/latest):
   - [Moddable Pong](https://github.com/inkandswitch/backstitch/releases/latest/download/moddable-pong-with-backstitch.zip)
   - [Moddable Platformer](https://github.com/inkandswitch/backstitch/releases/latest/download/moddable-platformer-with-backstitch.zip)
   - [Threadbare](https://github.com/inkandswitch/backstitch/releases/latest/download/threadbare-with-backstitch.zip)
2. Descomprime el archivo
3. Haz doble clic en el script de inicio correspondiente a tu plataforma:
   - Windows: `backstitch-launcher.windows.exe`
   - Linux: `backstitch-launcher.linux`
   - macOS: `backstitch-launcher.macos`

<div class="warning">

### Nota

Tanto Moddable Platformer como Threadbare utilizan el [servidor de sincronización alfa](../../server/alpha-server.md) de Backstitch. Por favor, no dependas de este servidor para almacenar tus datos, ya que podríamos necesitar reiniciarlo en cualquier momento; asegúrate de tener siempre una copia local de tu proyecto. Además, cualquier persona que conozca el ID de tu proyecto podrá acceder a él, así que no almacenes datos sensibles ni propietarios.

</div>
