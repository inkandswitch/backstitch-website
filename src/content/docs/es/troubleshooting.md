# Solución de problemas

Backstitch sigue siendo experimental y pueden producirse errores.

Mientras guardes tu trabajo con regularidad, es poco probable que pierdas datos. Todos tus cambios se guardan en el sistema de archivos local de tu equipo.

Si la sincronización no funciona, puedes recurrir a otros métodos para sincronizar datos, como un repositorio de Git.

Si Godot se cuelga o deja de responder, prueba a reiniciar el editor.

## ¡He encontrado un error!

Por favor, abre una incidencia en nuestra [página de GitHub](https://github.com/inkandswitch/backstitch/issues) o escríbenos a <a href="mailto:paul@inkandswitch.com">paul@inkandswitch.com</a>. ¡Procura incluir pasos detallados para reproducir el error en tu informe!
