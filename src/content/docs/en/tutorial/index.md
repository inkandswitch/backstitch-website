# Tutorial

<div class="warning">

### Important!

As you use our plugin, make sure to have Godot open and Backstitch loaded when you make changes to any files in your project. There's currently a bug where, in some rare cases, work done while Godot/Backstitch is closed can be overwritten by other team members without the ability to revert.

Soon, we will be moving to a hash-based local filesystem index in order to prevent this and handle your changes while Godot is closed.

In the meantime, remember to have Backstitch running while making changes, and always keep backups!

</div>

With Backstitch and the correct version of Godot installed, you're ready to begin!
